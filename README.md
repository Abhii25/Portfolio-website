## ?? Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for testing)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Abhii25/Portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Portfolio
   ```

3. **Open in browser**
   
   Simply open `index.html` in your web browser, or use a local server:
   
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (npx)
   npx serve
   ```

4. **View the site**
   
   Open `http://localhost:8000` in your browser.

## ?? Customization

### Changing Colors

The main color scheme is defined using CSS custom properties in `css/style.css`:

```css
:root {
    --bg-dark: #0a0a0f;
    --bg-card: rgba(15, 15, 25, 0.8);
    --accent-primary: #00d9ff;
    --accent-secondary: #00ff88;
    --text-primary: #ffffff;
    --text-muted: #8892b0;
}
```

### Updating Content

- **Personal Information**: Edit `index.html` to update name, bio, and contact details
- **Profile Image**: Replace `assets/profile.jpg` with your own image
- **Logo**: Replace `assets/logo.png` with your custom logo
- **Resume**: Replace `assets/resume.pdf` with your updated resume

### Contact Form

The contact form is integrated with [Formspree](https://formspree.io/). To use your own endpoint:

1. Create a Formspree account
2. Set up a new form
3. Replace the form action URL in `index.html`:
   ```html
   <form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
   ```
